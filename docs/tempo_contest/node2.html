Infrastructure

Two computers were used: Alonso and Uhle algorithms were run on Windows OS, the rest on Linux OS, both 1.6GHz, with 512 MB RAM

The evaluation framework was designed as a set of matlab, perl, shell and dos scripts. Distortions were added to the signal with the help of the sofwares Sox and Matlab. (All these files are available here.)
